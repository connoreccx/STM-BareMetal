## üìò Course Overview

The course teaches the fundamentals of embedded systems by directly manipulating microcontroller registers using C.  
Topics covered include:

- Understanding microcontroller architecture  
- Memory maps and the ARM Cortex-M core  
- Clock configuration and system startup  
- GPIO configuration at the register level  
- Timers, interrupts, and NVIC  
- UART communication without HAL  
- Writing linker scripts  
- Working with startup files and vector tables  
- Building bare-metal projects from scratch

The goal is to develop a solid foundation for writing efficient, low-level firmware that gives full control over hardware.
